# Tecnologias utilizadas


## Back end
 - Node js
 - Typescript
 - postgresql
 - Express 
 - docker
 - docker-compose

## front end
  - React
  - style sheet
  - react-icons
  - react-toastify
  - axios

## mobile
  - React Native
  - Expo
  - axios

# Como executar o projeto
## Back end
pre-requesito: docker
```bash
 # Clonar o repositório
 git clone https://github.com/Fabricio1805/Projeto-Pizzaria.git
 
 # Entrar na pasta do projeto
 cd Projeto-Pizzaria 
 
 # Executar o comando docker
 docker-compose up
```

## Front end

Pré-requisitos: npm / yarn

```bash
# Clonar o repositório
git clone https://github.com/Fabricio1805/Projeto-Pizzaria.git

# Entrar na pasta do projeto
cd Projeto-Pizzaria

# Entrar na pasta do projeto front end
cd frontend

# Instalar dependências
yarn

# Executar o projeto
yarn dev

```

## Mobile
Pré-requisitos: expo

```bash
# Clonar o repositório
git clone https://github.com/Fabricio1805/Projeto-Pizzaria.git

# Entrar na pasta do projeto
cd Projeto-Pizzaria

# Entrar na pasta do projeto front end
cd mobile

# Instalar dependências
npm i

# Executar o projeto
npm start

```
